Counter App — Professional Version

Open `index.html` in the browser or navigate to `/JS-Projects/Counter-App/` when running the app.

Features:
- Single source of truth state in JavaScript
- Step control (+/-), min/max bounds with validation
- Persistent state in localStorage
- History tracking with timestamps and export/clear
- Keyboard support: ArrowUp/ArrowDown for inc/dec, Ctrl/Cmd+0 to reset

Files:
- index.html — page
- script.js — vanilla JS implementation

Notes:
- Uses Tailwind CDN for styling; no build step required for this project page.
